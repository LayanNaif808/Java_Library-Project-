# Library Management System (Java Project 1)

This is a simple console-based application built to manage a small library registry using Java. It satisfies all the requirements for Java courses 101 to 104 on Satr Platform.

## Features Included
- **OOP Principles:** Built using the `Book` class to instantiate objects.
- **Data Structures:** Uses `ArrayList` to dynamically store and manage book logs.
- **Interactive Menu:** Fully automated menu for adding, displaying, searching, and deleting entries.
- **Exception Handling:** Implements `try-catch` blocks to protect user input from invalid types.

## How to Run
1. Type the code into any Java Compiler.
2. Run the `LibraryManagementSystem` class.
3. Follow the interactive menu on screen.
